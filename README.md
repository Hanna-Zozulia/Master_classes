# Сайт с мастер-классами

Это веб-приложение для просмотра и управления мастер-классами, разработанное на PHP в процедурном стиле с элементами архитектуры MVC.

* Пользователь может:
* Просматривать список доступных мастер-классов.
* Фильтровать мастер-классы по категориям.
* Просматривать подробную информацию о каждом мастер-классе.
* Читать отзывы.
* Администратор может управлять контентом сайта через защищённую админ-панель.

## Описание

### Мастер-классы

* Отображение списка мастер-классов.
* Просмотр мастер-классов по категориям.
* Страница с детальной информацией о мастер-классе.

### Категории

* Группировка мастер-классов по категориям.
* Удобная навигация по каталогу.

### Админ-панель

* Авторизация администратора.
* Добавление новых мастер-классов.
* Редактирование существующих мастер-классов.
* Удаление мастер-классов.
* Управление категориями.

### Пользовательский интерфейс

* Простая и понятная структура сайта.
* Отдельный интерфейс для администратора.
* Использование HTML и CSS для отображения данных.

## Структура проекта

*   `/` - Корневая директория публичной части.
*   `/Admin` - Корневая директория панели администратора.
*   `/controller` и `/Admin/ControllerAdmin` - Файлы с логикой приложения (контроллеры).
*   `/model` и `/Admin/modelAdmin` - Файлы для работы с базой данных (модели).
*   `/view` и `/Admin/viewAdmin` - Файлы с HTML-разметкой (виды).
*   `/inc/Database.php` - Класс для подключения и работы с базой данных.
*   `/route` и `/Admin/routerAdmin` - Файлы, отвечающие за маршрутизацию (роутинг).
*   `/public` - Статичные файлы: CSS, JavaScript, шрифты.
*   `masterclass.sql` - Дамп базы данных для первоначальной настройки.

## Используемые технологии

- PHP (MVC)
- MySQL
- HTML5
- CSS3

## Установка и запуск

1.  **База данных:**
    -   Создайте новую базу данных в вашей СУБД (MySQL).
    -   Импортируйте в нее файл `masterclass.sql`.

2.  **Настройка подключения:**
    -   Откройте файл `inc/Database.php`.
    -   Внесите данные для подключения к вашей базе данных в конструкторе класса `Database` (хост, имя пользователя, пароль, имя БД).

3.  **Веб-сервер:**
    -   Настройте ваш веб-сервер (например, Apache) так, чтобы корневая директория сайта указывала на папку с этим проектом.

4.  **Готово!**
    -   Откройте сайт в браузере.

## Панель администратора

- **URL:** `http://localhost:3000/Admin/`
- **Учетные данные:** Для входа используйте логин и пароль, которые хранятся в таблице `users` (или аналогичной) в вашей базе данных.
